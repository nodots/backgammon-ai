# Release Notes - Version 3.5.0

## 🎯 Revolutionary Plugin Architecture

Version 3.5.0 introduces a **groundbreaking plugin system** that transforms nodots-backgammon-ai into an **open, extensible platform** for backgammon AI development. This release emphasizes **openness, community contribution, and algorithmic diversity**.

---

## 🚀 Major Features

### 🔌 **Plugin System for Move Analyzers**

- **Complete plugin architecture** with dynamic loading capabilities
- **Five built-in analyzers** covering different strategies and use cases
- **Hot-swappable AI engines** - switch between different analyzers seamlessly
- **Context-aware analysis** with board state and position ID support
- **Template-based development** for easy plugin creation

### 🧠 **Nodots AI Foundation**

- **New NodotsAIMoveAnalyzer** - The nascent AI model with strategic heuristics
- **Three-tier strategy system**:
  1. **Safety First**: Prioritizes defensive moves and blot protection
  2. **Offensive Play**: Focuses on attacking and blocking opportunities
  3. **Racing Strategy**: Optimizes checker advancement and bearing off
- **Extensible framework** for developing sophisticated AI strategies

### 🌐 **Open Development Platform**

- **Plugin loader** automatically discovers and loads custom analyzers
- **Standardized MoveAnalyzer interface** for consistent plugin development
- **Community-friendly architecture** encouraging third-party contributions
- **TypeScript-first design** with full type safety and IntelliSense support

## 💡 Plugin Development

### **Creating Custom Analyzers**

```typescript
import { MoveAnalyzer, MoveAnalyzerContext } from '@nodots-llc/backgammon-ai'

export class MyCustomAnalyzer implements MoveAnalyzer {
  async selectMove(
    moves: BackgammonMoveBase[],
    context?: MoveAnalyzerContext
  ): Promise<BackgammonMoveBase | null> {
    // Your custom AI logic here
    return moves[0] || null
  }
}

export default MyCustomAnalyzer
```

### **Dynamic Plugin Loading**

```typescript
import { loadAnalyzersFromPluginsDir } from '@nodots-llc/backgammon-ai'

const analyzers = loadAnalyzersFromPluginsDir('./my-plugins')
const move = await analyzers['myCustomAnalyzer'].selectMove(moves, context)
```

## 🎨 Enhanced API Surface

### **New Exports**

- `MoveAnalyzerContext` for context-aware analysis
- `loadAnalyzersFromPluginsDir` for dynamic plugin loading
- `NodotsAIMoveAnalyzer` - The new AI foundation
- `selectMoveFromList` - Analyzer-agnostic move selection

### **Improved Integration**

```typescript
import {
  NodotsAIMoveAnalyzer,
  GnubgMoveAnalyzer,
  RandomMoveAnalyzer,
  selectMoveFromList,
} from '@nodots-llc/backgammon-ai'

// Use different analyzers based on game context
const analyzer =
  difficulty === 'expert' ? new GnubgMoveAnalyzer() : new NodotsAIMoveAnalyzer()

const bestMove = await selectMoveFromList(moves, analyzer)
```

## 📝 Migration Guide

**From v3.4.x to v3.5.0:**

- No breaking changes to existing APIs
- New plugin system is opt-in
- Enhanced move selection with `selectMoveFromList`
- Context-aware analysis available for advanced use cases

**Example Migration:**

```typescript
// Before (v3.4.x)
const move = moves[Math.floor(Math.random() * moves.length)]

// After (v3.5.0) - More intelligent selection
import {
  selectMoveFromList,
  NodotsAIMoveAnalyzer,
} from '@nodots-llc/backgammon-ai'
const move = await selectMoveFromList(moves, new NodotsAIMoveAnalyzer())
```
